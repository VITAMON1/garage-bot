# 🚗 Гараж Мечты — Бот для сбора средств

Бот для телеграм-канала, помогающий собирать пожертвования на покупку гаража в Екатеринбурге.

## 🚀 Запуск

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте `.env` по образцу `.env.example`
4. Запустите: `python bot.py`

## 📁 Структура
- `bot.py` — основной код
- `database.py` — работа с SQLite
- `.env` — переменные окружения (токен, ID админа)

## ⚠️ Важно
Все переводы — добровольные пожертвования (ст. 582 ГК РФ).
